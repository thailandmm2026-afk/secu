
# DEV-KiKi Bot

Telegram Python & Node.js 24/7 Hosting Bot

## Features

- 🐍 Python 24/7 Runner
- 🟨 Node.js 24/7 Runner
- 📦 Auto pip Install
- 📦 Auto npm Install
- 🔐 Security Scan
- 💎 Premium System
- 🔑 Subscription Key System
- 👥 Admin Panel
- 📊 System Status
- 📁 SQLite Database
- 🌐 Flask Keep Alive
- ☁️ Deploy on Render / Railway

---

## Requirements

- Python 3.10+
- pip
- Node.js (Optional)

---

## Installation

```bash
git clone [[https://](https://github.com/thailandmm2026-afk/secu.git
cd REPOSITORY

pip install -r requirements.txt
```

---

## Environment Variables

Create the following variables.

```
BOT_TOKEN=YOUR_BOT_TOKEN
OWNER_ID=7308292609
ADMIN_ID=7308292609
ADMIN_USERNAME=@kiki20251
```

---

## Run

```bash
python secu.py
```

---

## Deploy Render

Build Command

```bash
pip install -r requirements.txt
```

Start Command

```bash
python secu.py
```

---

## Deploy Railway

Railway automatically installs packages from `requirements.txt`.

Start Command

```bash
python secu.py
```
